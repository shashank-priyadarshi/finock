To use this static PDF library web page:

1. Create a folder named 'pdfs' in this directory (next to index.html).
2. Place your PDF files in the 'pdfs' folder. For example:
   - pdfs/sample1.pdf
   - pdfs/sample2.pdf
   - pdfs/sample3.pdf
   - pdfs/sample4.pdf
3. You can rename the files and update the display names in index.html as needed.
4. Open index.html in your browser to use the PDF viewer.

Note: The web page will not display PDFs unless they are present in the 'pdfs' folder.
